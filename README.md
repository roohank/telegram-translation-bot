این ربات تلگرامی با استفاده از **Pyrogram** و **Deep Translator** ساخته شده و امکان ترجمه بین **۱۳۳ زبان زنده دنیا** را فراهم می‌کند.  

در اولین اجرای ربات، شما یک‌بار **زبان مبدأ** و **زبان مقصد** را انتخاب می‌کنید. این تنظیمات در یک فایل JSON ذخیره می‌شوند و تا زمانی که آن‌ها را تغییر ندهید، به‌عنوان **تنظیمات پیش‌فرض** استفاده خواهند شد.  

---

This Telegram bot is built using **Pyrogram** and **Deep Translator**, allowing translation between **133 world languages**.  

When you run the bot for the first time, you will **set the source and target languages once**. These settings are saved in a **JSON file** and will remain as the **default languages** until you change them.  

---

## ✨ ویژگی‌ها | Features  
✅ پشتیبانی از **۱۳۳ زبان** | Supports **133 languages**  
✅ انتخاب زبان مبدأ و مقصد در اولین اجرا | Set source and target languages on first run  
✅ ذخیره تنظیمات در فایل JSON | Automatically saves settings in a JSON file  
✅ رابط کاربری ساده و کاربرپسند | Simple and user-friendly Telegram interface  
✅ سرعت بالا و استفاده از کتابخانه‌های قدرتمند Python | Fast and efficient with powerful Python libraries

## 🛠 تکنولوژی‌های استفاده‌شده | Built With  
- [Pyrogram](https://docs.pyrogram.org/) – Telegram API Client  
- [Deep Translator](https://pypi.org/project/deep-translator/) – Translation Library

## 🚀 نحوه استفاده | How to Use  
1️⃣ ربات را اجرا کنید | Run the bot  
2️⃣ در اولین اجرا، زبان‌ها را انتخاب کنید | Choose source and target languages on first run  
3️⃣ متن خود را ارسال کنید تا ترجمه شود | Send any text to get it translated  
4️⃣ هر زمان بخواهید می‌توانید زبان‌ها را تغییر دهید | You can change the languages anytime
